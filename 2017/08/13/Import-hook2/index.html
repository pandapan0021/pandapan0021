flask 中使用场景

flask作为轻型框架，依赖大量的外部拓展来提供额外的功能。
旧式风格的扩展包的命名方式为flaskext.(somename),新式的包名为flask_(somename)。
为了达到兼容的目的flask提供了使用Importer Protocol机制的解决方案。
当在flask应用中使用flask.ext.(somename)时，flask可以查找flaskext.(somename)与
flask_(somename)来确定应该导入哪个包。

源代码中的位置为 flask/exthook.py 和 flask/ext/__init__.py

具体实现

从Importer protocol中看出要导入flask.ext.something的时候首先会导入flask.ext包，

而在flask.ext包中flask首先导入了一个集查找器和载入器于一身的导入器。

from ..exthook import ExtensionImporter

之后实例化并安装该导入器。

importer = ExtensionImporter(['flask_%s', 'flaskext.%s'], \_\_name\_\_)
importer.install()

之后就是执行在flask.ext包中导入子模块的过程了，

根据Importer protocol，首先调用find_module函数。

def find_module(self, fullname, path=None):
    if fullname.startswith(self.prefix) and \
       fullname != 'flask.ext.ExtDeprecationWarning':
        return self

可见find_module并没有做什么事情，只是在确认导入的全名以 flask.ext为前缀时，
将importer作为载入器返回，等待调用其的load_module方法。

load_module的实现主要部分为：

for path in self.module_choices:
    realname = path % modname
    try:
        __import__(realname)
    except ImportError:
        ....

        continue

其中的 self.module_choices 为实例化时候传入的[‘flaskext.%s’, ‘flask_%s’]。

hack

我们修改flask.ext中的源码 在module_choices中增加一个flask_extension_%s来测试一下
效果。

importer = ExtensionImporter(['flask_%s', 'flaskext.%s', flask_extension_%s'], __name__)

之后我们写一个flask_extension_whatever模块。

flask_extension_whatever.py

class WhatEver(object):
    def __init__(self, app=None):
        if app:
            self.init_app(app)

    def init_app(self, app):
        app.extensions['whatever']= self
        self.name = app.config['WHATEVER_NAME']

    def say_hello(self, name=None):
        _name = name or self.name
        if _name:
            print("hello %s" % _name)
        else:
            print("hello")

setup.py

from setuptools import setup

setup(
name='flask-extension-whatever',
version='0.0.1',
py_modules=['flask_extension_whatever']
)

之后安装该模块

pip install .

在我的vps上(python3.6, centos6, virtualenv 15.1.0)使用 python setup.py install默认会安装egg
格式的文件，但是python却无法识别出已安装模块，顾使用pip安装。

app.py

from flask import Flask
from flask.ext.whatever import WhatEver

app = Flask(__name__)
app.config['WHATEVER_NAME'] = 'john'

WE = WhatEver(app)

WE.say_hello()
