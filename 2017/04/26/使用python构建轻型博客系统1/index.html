使用python构建轻型博客系统1

Veröffentlicht am 2017-04-26

分解：

博客系统主要有三个部分组成

+----------+        +------------+    +---------+
|  写作    |        |  页面生成  |    |  部署   |
+----------+        +------------+    +---------+
     |                    |                |
  标记语言             页面主题         github/
     |                    |              vps
  解析器               模板系统

现在大多数的标记语言使用的是markdown，虽然简单强大，但是对于python程序员来说最
熟悉的还是reStructText。RST已经成了python的准官方语言。所以我们使用RST作为后端
的写作语言。

模板系统我们选用jinja2

部署到github使用GitPython，vps通过ftplib使用ftp协议进行传输

reStructText

最好的RST解析器当然是docutils

主题

查询配置中设置的主题文件夹，将此文件夹作为模板系统的主工作目录，在通过模板生成
页面文件写入待部署文件夹之后将静态文件拷贝到待部署文件夹。

部署

对于github直接将仓库push即可
对于vps则删除所有远端文件后，将本地文件全部上传。
